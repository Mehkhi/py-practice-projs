## Sprite File Information

All enemy sprites are **PNG files** located in `assets/sprites/`

### File Format
- **File Type:** PNG (Portable Network Graphics)
- **Expected Format:** Transparent PNG with alpha channel
- **Recommended Size:** 128x128 pixels (or larger, will be scaled)
- **Color Depth:** 32-bit RGBA (with transparency)

### Directory Structure
All enemy sprites should be placed in:
```
assets/sprites/{sprite_id}.png
```

### Download Instructions

1. Each enemy requires a PNG file named `{sprite_id}.png`
2. Files should be placed in: `assets/sprites/`
3. Use the provided download script: `download_enemy_sprites.py`

To use the download script:
```bash
# Edit download_enemy_sprites.py and set your DOWNLOAD_URL
# Example: DOWNLOAD_URL = 'https://example.com/sprites/{filename}'
.venv/bin/python3 download_enemy_sprites.py
```
