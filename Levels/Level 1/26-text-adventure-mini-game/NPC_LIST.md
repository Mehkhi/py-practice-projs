## Sprite File Information

NPCs require **two types of sprite files**:

1. **Main Sprite**: PNG file in `assets/sprites/` (for overworld display)
2. **Portrait**: PNG file in `assets/sprites/portraits/` (for dialogue/UI)

### File Format
- **File Type:** PNG (Portable Network Graphics)
- **Expected Format:** Transparent PNG with alpha channel
- **Recommended Size:**
  - Main sprites: 128x128 pixels (or larger, will be scaled)
  - Portraits: 64x64 or 128x128 pixels (for dialogue boxes)
- **Color Depth:** 32-bit RGBA (with transparency)

### Directory Structure
```
assets/sprites/{sprite_id}.png                    # Main sprite
assets/sprites/portraits/portrait_{name}.png     # Portrait
```

### Download Instructions

1. Each NPC requires a sprite PNG: `{sprite_id}.png` in `assets/sprites/`
2. Each NPC requires a portrait PNG: `portrait_{name}.png` in `assets/sprites/portraits/`
3. Use the provided download script: `download_npc_sprites.py`

To use the download script:
```bash
# Edit download_npc_sprites.py and set your download URLs
# Example:
#   SPRITE_DOWNLOAD_URL = 'https://example.com/sprites/{filename}'
#   PORTRAIT_DOWNLOAD_URL = 'https://example.com/portraits/{filename}'
.venv/bin/python3 download_npc_sprites.py
```
