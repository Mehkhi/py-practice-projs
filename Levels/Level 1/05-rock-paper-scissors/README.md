# Rock-Paper-Scissors

A simple command-line Rock-Paper-Scissors game. Play against the computer, validate player input, and keep track of the running score until you choose to quit.

## How to Run

```bash
python rock_paper_scissors.py
```

## Example Session

```
Welcome to Rock-Paper-Scissors!
Choose rock, paper, or scissors (or type 'quit' to exit): rock
You chose rock, computer chose scissors.
You win this round!
Player: 1 | Computer: 0 | Ties: 0
-
Choose rock, paper, or scissors (or type 'quit' to exit): q
Thanks for playing!
Final score:
Player: 1 | Computer: 0 | Ties: 0
```
