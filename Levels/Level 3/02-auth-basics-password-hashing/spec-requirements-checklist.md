# Auth Basics & Password Hashing - Spec, Requirements, and Checklist

## Project Specification
Implement login and logout functionality using salted password hashing and session management for secure user authentication.

## Requirements (Core Skills)
- werkzeug.security for password hashing
- Session management for maintaining user state
- Secure cookies to protect session data

## Completion Checklist

### Milestones
- [ ] Implement password reset functionality using tokens
- [ ] Add email flow for password resets (in development mode)

### Stretch Goals
- [ ] Implement rate limiting to prevent brute-force attacks
- [ ] Add account lockout mechanism after failed attempts
